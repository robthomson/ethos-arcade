# Dojo (Ethos)

System tool entrypoint:

- `main.lua`

Controls:

- Aileron/Roll stick: move left/right
- Rudder/Yaw stick: punch
- Elevator/Pitch stick: jump
- `Enter` (short): start round / restart after KO
- `RTN` / `Exit`: return to arcade menu

Settings:

- Settings menu available from the home screen
- Difficulty (Easy / Normal / Hard)

Settings are stored in:

- `retrofight.cfg` (local file in the script folder)
