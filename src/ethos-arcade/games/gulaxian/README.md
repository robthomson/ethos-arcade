# GaLuaxian (Ethos)

System tool entrypoint:

- `main.lua`

Controls:

- Aileron/Roll stick: move left/right
- Elevator/Pitch stick: move up/down
- `Enter` (short): start game
- `Enter` (long, on home screen): open settings form
- `RTN` / `Exit`: close settings form or end current run

Settings and best score are stored in:

- `gulaxian.cfg` (local file in the script folder)

Settings include:

- quality/performance toggles
- gameplay toggles
- background selection (`back.png`, `back2.png`, `back3.png`)
